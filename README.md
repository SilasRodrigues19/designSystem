## Getting Started

#### Pre requisites


<br>

1. Clone the repo

```sh
  git clone https://github.com/SilasRodrigues19/designSystem.git && cd designSystem
```

2. Install NPM packages

```sh
  npm i
```

3. Run the project

```sh
  npm run dev
```

4. Run Storybook
```sh
  npm run storybook
```
